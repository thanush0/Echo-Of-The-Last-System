# Release Notes

## Version 1.0.0 - Initial Release

**Release Date:** January 2025

### Status
Feature-complete and stable. This is a finished product, not early access.

### What's Included

**Core Game:**
- Complete story with five distinct endings
- Turn-based combat system with five enemy types
- NPC dialogue system with The Oracle
- Quest system with main quest and side quests
- Save/load functionality (3 slots)
- Full stat progression and skill unlocking

**Interfaces:**
- CLI (terminal-based, zero dependencies)
- Basic Tkinter GUI (simple, built-in)
- Advanced Tkinter GUI (modern design, animations)
- Pygame GUI (visual effects, requires pygame)

**Features:**
- Six discoverable lore fragments
- Multiple dialogue branches
- Hidden Luck stat
- Corruption system affecting endings
- Action-based stat growth
- System glitch mechanics (15+ types)
- Enemy evolution system
- Quest mutation mechanics

### Known Issues

- Some antivirus software may flag .exe files (PyInstaller false positive)
- Pygame GUI requires build tools on some systems (use Tkinter GUIs instead)

### Installation

See README.md for complete installation instructions.

### Testing

- All core systems tested
- All five endings confirmed accessible
- Save/load tested across all slots
- No known game-breaking bugs
