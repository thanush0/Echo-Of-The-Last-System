# Complete Walkthrough - Echo of the Last System

## First Playthrough Guide

### Getting Started

Launch the game and select "NEW GAME". You'll go through:
1. System initialization sequence
2. Character designation (choose Unknown, Custom, or Randomized)
3. Opening narrative

### First 10 Minutes

**Recommended Actions:**
1. Explore 2-3 times to understand event types
2. Engage in your first combat (likely a Corrupted Slime)
3. Use Analyze to learn combat mechanics
4. Rest when HP drops below 50%
5. Check Status to see stat growth
6. Save your progress

### Combat Strategy

**Enemy Types:**
- **Corrupted Slime** (Level 1) - Weak, good for practice
- **Ruins Skeleton** (Level 2) - Medium difficulty, drops Memory Shards
- **Glitched Wolf** (Level 3) - Fast and dangerous
- **System Wraith** (Level 3+) - Rare, powerful
- **Corrupted Guardian** (Level 5+) - Boss, drops Core Fragment

**Combat Tips:**
- Analyze new enemies first
- Watch your HP carefully
- Fleeing is valid at low HP
- Skills cost MP but deal more damage
- Critical hits depend on hidden Luck stat

### The Oracle

**When:** Appears randomly during exploration (5% chance)

**First Meeting Choices:**
1. "How do you know my name?" - Learn about memory wipe (+2 INT)
2. "What happened to the others?" - Learn previous users' fates (+2 INT, lore flag)
3. "Why should I trust you?" - Philosophical response (+1 INT)
4. "Tell me the truth" - Major lore reveal (+3 INT, +10 corruption)
5. [Remain silent] - Wise choice (+2 LUCK)

**Recommendation:** Choose option 2 or 4 for maximum story understanding.

### Quest Guide

**Main Quest: "Find the System Core Fragment"**

Objectives:
1. Explore the Forgotten Ruins (5 times)
2. Defeat the Corrupted Guardian
3. Obtain a System Core Fragment

**Strategy:**
- Level up to 5+ before seeking the Guardian
- Guardian appears randomly at higher exploration counts
- Appears as rare combat at level 5+
- Recommended HP: 150+

### Ending Requirements

**Ending 1: Survival**
- Reach level 10
- Do NOT find Core Fragment
- Just survive

**Ending 2: System Takeover**
- Corruption ≥ 80%
- Have System Core Fragment
- Use forbidden skills frequently

**Ending 3: World Collapse**
- System integrity ≤ 0%
- Use Reality Tear skill repeatedly
- Experience many anomalies

**Ending 4: Godless Freedom**
- Corruption ≤ 30%
- Have System Core Fragment
- Met Oracle
- Level 7+

**Ending 5: True Ending**
- Have System Core Fragment
- Collect 5+ Memory Shards
- Learned truth (dialogue flag)
- Met Oracle
- Level 8+
- Corruption: 30-60% (balanced)

### Advanced Tips

1. **Save Scumming:** Use multiple save slots to explore different paths
2. **Stat Growth:** Diverse actions = better overall stats
3. **Corruption Balance:** 30-60% is ideal for True Ending
4. **Memory Shards:** Fight Ruins Skeletons to collect
5. **Luck Discovery:** Reaches 50% corruption to reveal the hidden stat
